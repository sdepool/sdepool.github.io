ANSYS 19.0 安装步骤

方法一（推荐）

破解文件同方法二不一样！

  1. 安装 ANSYS Products 19.0 Win64 （管理员权限）
    不要安装ANSYS License Manager！！！！
    当需要输入license server name时请直接输入”localhost” (不要写自己的电脑名！)
  2. 复制”Shared Files” 到ANSYS相应的安装目录，覆盖原有文件
    (默认为C:\Program Files\ANSYS Inc)
  3. 破解成功!

方法二（需要安装ANSYS License Manager）

  1. 安装 ANSYS 19.0, 需要输入license server name时填写host名称
  2. 安装 ANSYS 19.0 License Manager, 不要安装 license file.
  3. 把ansys190.rar 中文件拷贝并覆盖到”Program Files\ANSYS Inc\Shared Files\Licensing\winx64\”中

  4. 用 A1190_Calc.exe 生成 license file，正确步骤：

    1. A190_Calc.exe双击打开后，Y/N输入：N；
    2. 然后name：输入自己电脑名；
    3. 然后输入自己电脑的物理地址，licence manager 里点击Get System Hostid Information，有三个选项，下面是两个MAC地址，如果你用网线连电脑的话，网卡的MAC地址就是你的IDGet System Hostid Information有三个选项，下面是两个MAC地址，如果你用网线连电脑的话，网卡的MAC地址就是你的ID
  5. 打开 License Management Center
  6. 点击 “Add a License File” ，安装刚生成的文件

ANSYS 卸载步骤

  1. 卸载当前的ansys和license manager（如有）
  2. 删除ansys有关环境变量（系统变量）
  3. regedit 搜索 “license” ，删除ansys有关项
  4. 删除有关进程
  5. 删除剩余文件夹
